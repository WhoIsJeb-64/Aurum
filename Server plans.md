# Ranks

## Player Ranks
The normal player ranks. Players rank up by reaching certain playtime milestones.
### <span style="color:#555555">Wanderer:</span>
The default player rank.\
Can set up to **4** homes.

### <span style="color:#00AA00">Dweller:</span>
Obtained by reaching **30** hours of playtime.
* Can build at spawn.
* Can place lava and tnt.
* Can send images and gifs in any discord channel.

Can set up to **5** homes.

### <span style="color:#55FF55">Elder:</span>
Obtained by reaching **100** hours of playtime.
* Can use chat colors.
* Can set their own nickname.

Can set up to **7** homes.

### <span style="color:#00AAAA">Ancient:</span>
Obtained by reaching **500** hours of playtime.
* Can set their own join and leave messages.
* Has access to /hat and the logblock tool.

Can set up to **10** homes.

## Donator Ranks
Players can obtain a donator rank by donating ≥$7 to the server, or\
by boosting the discord server, while they are boosting.
### <span style="color:#FF55FF">Patron:</span>
Obtained by donating ≥$7 to the server as a Wanderer.
* Can send images and gifs in any discord channel.
* Can use chat colors.
* Can set their own nickname.

Can set up to **5** homes.

### <span style="color:#55FFFF">???:</span>
Obtained by donating ≥$7 to the server as a Dweller, or\
reaching **30** hours of playtime as a Patron.
* Can build at spawn.
* Can place lava and tnt.
* Can set their own join and leave messages.

Can set up to **7** homes.

### <span style="color:#FFAA00">Noble:</span>
Obtained by donating ≥$7 to the server as an Elder, or\
reaching **100** hours of playtime as a ???.
* Has access to /hat and the logblock tool.

Can set up to **9** homes.

### <span style="color:#FFFF55">Mystic:</span>
Obtained by donating ≥$7 to the server as an Ancient, or\
reaching **500** hours of playtime as a Noble.
* Cosmetic aside from the increase in max homes.

Can set up to **12** homes.

## Staff Ranks
The rank ladder for members of the server staff team.
### <span style="color:#5555FF">Warden:</span>
Given to newly accepted members to the staff team.\
Responsible for enforcing the server rules and keeping order.
* Can use basic moderation commands:\
`/ban /kick /mute /warn /modview /invsee /vanish /tp /tphere /otp /whitelist`
* ^ Has any analogous permissions in the discord
* Has the perks of the Mystic rank.

Can set up to **12** homes.

### <span style="color:#AA00AA">Wizard:</span>
Given to people who develop plugins/bots/etc. for the server and are\
at least somewhat involved in the community.
* Functionally the same as the Warden rank.

Can set up to **12** homes.

### <span style="color:#AA0000">Admin:</span>
Reserved for the server founders.
* Has all permissions and console access.
* Has the perks of the Mystic rank.

Can set up to **12** homes.
